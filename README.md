# openclaw-langgraph-bridge
OpenClaw plugin bridging Kit-as-orchestrator with LangGraph workflows. Tools: langgraph_dispatch, langgraph_inspect. Webhook: status/milestone/decision/terminal/hitl event routing. State: managedFlows + enqueueSystemEvent. Per-Slack-thread isolation. POC for V1 product bot architecture (see DOPS-569 area, aidlc-fleet-poc design).
